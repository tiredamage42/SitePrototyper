# SitePrototyper

### An In-Browser `html` / `css` / `javascript` Text Editor

## [Try It Here!] (https://tiredamage42.github.io/SitePrototyper/)

- Uses [Ace](https://ace.c9.io "Visit the Ace homepage") for text highlighting

## How to Use:
- `Ctrl-S / Cmd-S` : Update View
